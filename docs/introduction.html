Introduction

21 April 2021 — aaduhuli

I am starting this blog to collect a random assortment of my thoughts, ideas and experiments. Previously I have used telegra.ph to write short excerpts that I shared with friends, this blog is a means to bring it all together in one place.

The blog is written in markdown, published using bashblog and hosted using github pages. The motivation behind this approach is to:

  1. Stay minimal.
  2. Create an independent, bloat-free corner of the world wide web.
  3. Learn how to best do all this.
